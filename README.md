# HowTheyVote - Data

This repository contains the complete contents of the [HowTheyVote.eu](https://howtheyvote.eu/) database and is updated at least weekly.

For more information, please take a look at the [main repo](https://github.com/HowTheyVote/epvotes).

## Usage

```
tar -cjvf dump-production.tar.bz2 /desired/path/for/dump.sql
```
